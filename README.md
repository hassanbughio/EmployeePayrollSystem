# Employee Payroll System

## Description
A Java-based Employee Payroll Management System with MySQL database.

## Features
- Employee CRUD
- Login System (Admin/User)
- Leave Management
- Payroll Calculation
- PDF Salary Slip Generation

## Technologies
- Java 19
- MySQL
- JDBC
- JasperReports

## Setup
1. Install Java 19
2. Install MySQL
3. Run `payroll_system.sql`
4. Update `DBConnection.java` with your credentials
5. Run `Main.java`
